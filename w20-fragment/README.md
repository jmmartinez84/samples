# W20 fragment sample

A W20 project demonstrating how to define and configure a fragment. Also demonstrates basic AngularJS and internationalization.

# Running it

    npm install
    bower install
    grunt connect

# Usage

Just point your favorite browser to the following URL:

    http://localhost:9001
